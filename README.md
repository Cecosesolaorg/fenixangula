# Tickera Fénix - Cecosesola 🚀 Prueba 

Aplicación web profesional diseñada para la gestión e impresión de tickets de precios para las Ferias de Cecosesola.

## 📋 Características
- **Modular y Organizada:** Código separado en funciones claras dentro de la carpeta `src/`.
- **Soporte Excel:** Carga masiva de productos desde archivos .xlsx.
- **Tickets Manuales:** Creación rápida de tickets individuales.
- **Tasa BCV:** Integración automática con el precio del dólar oficial.
- **Modo Oscuro:** Interfaz adaptable para mayor comodidad visual.
- **PWA (Progressive Web App):** Funciona sin conexión a internet una vez instalada en el navegador.

## 📁 Estructura del Proyecto (📂 src)
Para facilitar el mantenimiento, el código se ha dividido en:
- `datos_productos.js`: Listas de frutas, mascotas y charcutería.
- `subir_excel.js`: Lógica para procesar archivos de Excel.
- `gestion_tickets.js`: Funciones de impresión y creación de tickets.
- `precio_dolar.js`: Conexión con la API del BCV.
- `interfaz.js`: Manejo de botones y vistas.
- `estado.js`: Variables de control de la aplicación.
- `utilidades.js`: Formatos de fecha, moneda y temas.
- `principal.js`: Inicialización del sistema.

## 🚀 Instalación y Uso
1. Descarga o clona este repositorio.
2. Abre `index.html` en cualquier navegador moderno.
3. Para usar en GitHub Pages:
   - Sube los archivos a un repositorio de GitHub.
   - Ve a **Settings > Pages** y activa la publicación desde la rama `main`.
